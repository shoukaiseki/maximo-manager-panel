# 开发文档

## 1. 项目概述

### 1.1 项目简介
Maximo Manager Panel 是一个基于 Vue.js + Element UI 的 Maximo 元数据管理前端面板，用于查询和管理 MAXOBJECT、MAXATTRIBUTE、MAXRELATIONSHIP 等元数据。

### 1.2 技术栈
- **前端框架**: Vue.js 2.x
- **UI 组件库**: Element UI
- **构建工具**: Webpack
- **后端框架**: Solon（Java 17）
- **数据库**: DB2（Maximo）

## 2. 已实现功能模块

### 2.1 MaxObject 模块

#### 2.1.1 对象查询页面 (`/maxobject/index`)
- 对象名、描述模糊搜索
- 分页查询
- 点击行查看详情

#### 2.1.2 对象详情页面 (`/maxobject/detail/{objectname}`)
- 基本信息展示（对象名、描述、Java类等）
- 属性列表（可折叠、可搜索）
- 关系列表（可折叠、可搜索）
- 索引列表（可折叠，默认折叠）
- 域信息（可折叠）
- 子表关系（可折叠、可搜索）

**快捷键**: 在基本信息输入框中按 `Alt+F1` 查看字段属性详情

### 2.2 MaxAttribute 模块

#### 2.2.1 属性查询页面 (`/maximo/maxattr`)
- 对象名、字段名、字段描述搜索
- 支持 `=XXX` 精确搜索和 `%` 通配符模糊搜索
- 分页查询
- 点击行查看字段详情
- 右上角"显示所有列"、"显示字段名"按钮

#### 2.2.2 字段详情弹窗
- 展示字段完整属性信息
- 四个标签页：字段精简、字段完整、对象精简、对象完整
- 使用 Monaco Editor 展示 JSON

### 2.3 Relationship 模块

#### 2.3.1 关系查询页面 (`/maximo/relationship`)
- 关键字、关系名、父对象、子对象搜索
- 支持 `=XXX` 精确搜索和 `%` 通配符模糊搜索
- 分页查询
- 右上角"显示所有列"、"显示字段名"按钮

**快捷键**: 在搜索输入框中按 `Alt+F1` 查看字段属性详情

#### 2.3.2 关系详情弹窗
- 展示关系完整属性信息
- 两个标签页：关系精简、关系完整
- 使用 Monaco Editor 展示 JSON
- description 字段自动从 MAXOBJECT 表查询中文描述

**快捷键**: 在详情字段输入框中按 `Alt+F1` 查看字段属性详情

### 2.4 ApiCaller 模块

#### 2.4.1 API 调用页面 (`/apicaller`)
- 项目管理（创建、列表、复制、删除、导入、导出）
- API 调用测试
- 支持 APIPost JSON 导入

**认证方式**: 使用 `X-API-Key` 请求头

### 2.5 测试模块

#### 2.5.1 Excel 测试页面 (`/test/excel`)
- 支持从 Excel 复制数据粘贴到表格
- 11 列（序号、A-K）
- 默认 50 行

## 3. 开发约定

### 3.1 快捷键约定

| 快捷键 | 功能 | 适用页面 |
|--------|------|----------|
| `Alt+F1` | 查看字段属性详情 | 所有搜索输入框、详情弹窗输入框 |

### 3.2 搜索约定

| 搜索方式 | 语法 | 示例 |
|----------|------|------|
| 精确匹配 | `=XXX` | `=ASSET` |
| 模糊匹配 | `%XXX` | `%ITEM%` |
| 简单模糊 | 直接输入 | `ITEM` |

### 3.3 组件约定

#### 3.3.1 SksTable 组件
```javascript
{
  ownerName: '表名',
  uniqueId: '唯一标识',
  showAllColumnButton: true,      // 显示"显示所有列"按钮
  showTablePropName: false,       // 是否显示字段名（英文）
  serverPagination: true,         // 服务端分页
  tableColumnList: [...]          // 列定义
}
```

#### 3.3.2 字段详情弹窗
使用 `this.$alert()` 方式展示，参考 [MaxObjectDetail.vue](file:///E:/gitwork/maximo-manager-panel/src/views/maximo/maxobject/MaxObjectDetail.vue) 的实现模式。

### 3.4 API 约定

#### 3.4.1 Solon API（后端）
- 基础路径: `/solonapi/`
- 认证头: `X-API-Key`
- 返回格式:
  ```json
  {
    "code": 200,
    "data": {},
    "message": "成功"
  }
  ```

#### 3.4.2 Maximo API（前端）
- 基础路径: `/maximo/`
- 认证头: `maxauth` 或 `X-API-Key`

### 3.5 字段展示约定

#### 3.5.1 字段详情展示
当需要为字段添加 `Alt+F1` 快捷键查看详情时：
1. 将字段值展示改为只读 `el-input`
2. 绑定 `@keydown.native="handleKeyDown($event, {prop: '字段名', label: '中文标签'})"`
3. `handleKeyDown` 方法会调用 `getMaxAttributeList` 查询字段属性并弹窗展示

#### 3.5.2 描述字段处理
- 对于需要展示中文描述的字段，调用 `getMaxObjectDescription(objectname)` 获取
- 优先使用 `descriptionCn`（中文），其次 `description`（英文）

## 4. 常用 API

### 4.1 MaxObject 相关

```javascript
// 查询对象列表
getMaxObjectList(objectname, keyword, pageNum, pageSize)

// 查询对象详情
getMaxObjectDetail(objectname)

// 查询对象描述（英文+中文）
getMaxObjectDescription(objectname)

// 查询对象域信息
getMaxObjectDomains(objectname)

// 查询对象子表关系
getMaxObjectChildRelationships(objectname)
```

### 4.2 MaxAttribute 相关

```javascript
// 查询属性列表
getMaxAttributeList({objectname, attributename, description, pageNum, pageSize})

// 导出数据库配置
exportDbConfig(objectName, ignoreDefVal)
```

### 4.3 Relationship 相关

```javascript
// 查询关系列表
getMaxRelationshipList({keyword, name, parent, child, pageNum, pageSize})

// 查询所有关系（不分页）
getAllMaxRelationships()
```

## 5. 代码规范

### 5.1 Vue 文件结构
```
<template>
  <!-- 页面结构 -->
</template>

<script>
export default {
  name: 'ComponentName',
  data() { return {} },
  methods: {},
  mounted() {}
}
</script>

<style lang="scss" scoped>
/* 样式 */
</style>
```

### 5.2 命名规范
- **组件名**: PascalCase（如 `MaxObjectDetail`）
- **变量名**: camelCase（如 `objectDescription`）
- **常量**: UPPER_CASE（如 `MAXRELATIONSHIP`）
- **API 函数**: camelCase（如 `getMaxObjectDetail`）

### 5.3 注释规范
- 不添加注释，除非必要
- 复杂逻辑添加简短说明

### 5.4 异常处理
- API 请求必须包含 `.catch()` 处理
- 使用 `this.$message.error()` 提示用户
- 使用 `this.$message.success()` 提示成功

## 6. 部署说明

### 6.1 前端部署
```bash
npm install
npm run build
```

### 6.2 后端部署
- 确保 JDK 17 环境
- 修改 `server/src/main/resources/app-hd.yml` 配置数据库连接
- 运行 `run-solon-server-hd.bat`

### 6.3 代理配置
前端通过 `/maximo/` 和 `/solonapi/` 代理到后端服务。

## 7. 常见问题

### 7.1 API 认证失败（401）
- 检查 `X-API-Key` 请求头是否正确
- 检查后端 `app.yml` 中 `solon.apiKey` 配置

### 7.2 搜索无结果
- 检查搜索语法是否正确
- 确认数据库中存在对应数据

### 7.3 Alt+F1 无响应
- 确认输入框已获得焦点
- 确认使用的是键盘左侧 Alt 键
- 确认浏览器没有拦截 F1 快捷键

### 7.4 分页不生效
- 确认 `serverPagination` 设置为 `true`
- 确认分页参数传递正确
